# 2048Web
本项目我们的目标是实现一个 2048 的网页版游戏，项目分三周进行。



提交方式相同，Pull Request 形式提交到本仓库。请注意：

- 提交的文件夹命名为你的名字
- HTML 文件命名为 **index.html**
